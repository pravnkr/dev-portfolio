About

A bit about
what I do

I lead backend engineering at dealsplus.io, a Private Equity platform where fund managers model ownership structures, run valuations, and manage deal data. My team of seven ships across a Java/Spring Boot and React stack, with MySQL at the core.

Alongside that, I've been going deep on applied AI — not as a side interest but as a core part of how I build. That means custom MCP server implementations, RAG systems with domain-specific knowledge, multi-agent workflows using LangGraph and DeepAgents, and in-app chatbots that understand product context and can act within it.

These days I move between Java and Python depending on what the problem calls for. Most of the AI work is Python. Most of the financial engine work is Java. Some of the more interesting problems sit right at the boundary.

DealsPlus MCP Server

Custom MCP implementation that exposes live deal, investor, and cap table data to external AI platforms — including Claude — with OAuth, client-scoped access control, and SSE transport.
